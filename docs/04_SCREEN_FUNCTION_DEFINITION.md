# 1. 문서 작성 기준

본 문서는 IA에서 정의한 화면 구조를 기준으로, 각 화면에서 필요한 기능을 정리한다.

상세 API, DB 컬럼, BLE UUID, ERD는 별도 문서에서 정의한다.

이 문서에서는 **화면 단위의 기능과 동작 기준**만 정리한다.

## 2.1 온보딩 / 권한 안내 화면


| 항목     | 내용                          |
| ------ | --------------------------- |
| 화면 목적  | 앱 사용 전 서비스 목적과 필수 권한을 안내한다. |
| 주요 사용자 | 전동 휠체어 이용자                  |
| 우선순위   | P0                          |
| 진입 시점  | 앱 최초 실행 시                   |


### 주요 기능


| 기능              | 설명                       |
| --------------- | ------------------------ |
| 서비스 소개          | 동행가드가 센서 기반 위험 감지 앱임을 안내 |
