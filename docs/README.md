# docs — 프로젝트 문서

## 이 폴더의 역할

개발·리뷰 시 단일 진실 공급원(SSOT)으로 사용합니다.

## 사용하는 개발 도구

- Markdown
- (선택) Mermaid, draw.io 링크

- PRD, IA, API 명세, ERD 초안 작성
- 기술 가이드 및 프로토타입 링크 정리
