# backend — Spring Boot API

## 이 폴더의 역할

인증, 비즈니스 로직, DB 영속화, **FastAPI AI 서버와의 서버 간 연동**을 담당합니다.
앱·관리자 웹의 유일한 API 진입점입니다.

## 사용하는 개발 도구

- Spring Boot
- Java 17
- Spring Data JPA
- PostgreSQL 또는 MySQL

## 초기 작업 예정 내용

- 멀티 모듈 또는 단일 모듈 Spring Boot 스켈레톤
- 엔티티·리포지토리·기본 REST API
- AI 서버 호출 클라이언트 및 결과 저장
