# app — 모바일 앱

## 이 폴더의 역할

Android 기반 DongHaeng Guard 사용자(보호자·피보호자)용 모바일 클라이언트입니다.
위치, 경로, 알림 등 기능은 **Spring Boot 백엔드 API**를 통해서만 제공받습니다.

## 사용하는 개발 도구

- Android Studio
- Kotlin
- Gradle (Android)

## 초기 작업 예정 내용

- 프로젝트 스캐폴딩 (Kotlin, 최소 SDK 정책)
- 백엔드 인증·API 연동 레이어
- 주요 화면 와이어프레임 연동 (docs 참조)
