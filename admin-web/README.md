# admin-web — 관리자 웹

## 이 폴더의 역할

운영·관리자용 웹 대시보드입니다. 모니터링, 설정, 사용자 관리 등은 **Spring Boot 백엔드 API**만 호출합니다.

## 사용하는 개발 도구

- React
- TypeScript
- Vite

## 초기 작업 예정 내용

- Vite + React + TypeScript 프로젝트 생성
- API 클라이언트 및 인증 플로우
- 대시보드·목록 화면 골격
